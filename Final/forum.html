The Generic Modder Forum

Welcome to The Generic Modder forum, please keep the discussions civil and have a good time! To comment, enter your username and email to comment on the forums.




If you have a question about a mod you can also use the forum to talk about it, be sure to put a tag on there the community can help you with you issue.
Here is a table that explains what each of the tags mean


Tag Description possible causes Difficulty rating
Tags are abbreviations for the issues and keep clutter to a minimum so that we are able to figure out what exactly you are having trouble with, and help you. This is an explanation of what the tag means. This is where the possible causes are mentioned and the first things you should check for. The difficulty rating on a scale of 1-5 with 1 meaning its a minor problem and can fixed in a few minutes and 5 meaning it will probably take multiple people and some time to figure out.
(The moderators and/or the people working on the problem will rate it)
tech a tech tag means you are having problems getting your mod to work and could use a little help, these can vary from some minor mistakes in your code to a core issue that requires massive portions of your code to be redone
  • You mistyped a line of code and it is causing issues
  • You didnt put the files in the right place when making the mod
  • You didnt put the files in the right place when downloading the mod
 1-5
visual A visual tag means that you are having visual problems with your game, usually verifying your data or reinstalling the game/mod will fix this issue
  • The mod is conflicting with other files and the game is confused about what should go where
  • The game or mod is messing up and just needs to be reinstalled to ensure all the files are there
 1-3
bug A bug tag means that you have found a bug within someones mod and mainly serves to let the developer of the mod know
  • The possible causes vary from game to game
  • In order for the developer to be able to fix it, tell them what you did and how you did it so they can replicate it
 1-5
